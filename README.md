# Reaction-Diffusion Dynamics of Economic Beliefs

This repository contains the code for the paper "Reaction-Diffusion Dynamics of Economic Beliefs: A Geometric Model for the Propagation of Financial Resistance" by Marc Daghar.

## Overview

The model simulates the propagation of financial beliefs (alignment vs. resistance to debt-based systems) using reaction-diffusion dynamics on 2D grids and complex networks.
